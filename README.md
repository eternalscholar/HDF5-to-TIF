# HDF5-to-TIF
This is a Google Colab Notebook to create georeferenced tif files from Global Fire Emissions Dataset (HDF5 file). The HDF5 files can be downloaded from this link https://www.geo.vu.nl/~gwerf/GFED/GFED4/
